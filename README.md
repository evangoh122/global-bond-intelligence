# global-bond-intelligence
End-to-end pipeline for collecting, parsing, and querying bond filings from SEC Edgar, EDINET, ESMA, and HKEX. Includes XBRL/PDF extraction, structure-aware chunking, and a RAG layer for intelligent retrieval and citation over regulatory fixed income data.

> This project is AI-assisted — developed in collaboration with [Claude Code](https://claude.ai/code).
